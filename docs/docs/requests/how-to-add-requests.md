# How to Add Requests

Follow the steps below to create a request in APIEase.**

**
**

**Adding a Request**

Navigate to the Requests admin page by clicking the APIEase Requests submenu in the lower left portion of your Shopify Admin.Click the Add Request button at the top right of the screen.Configure the [HTTP](https://apiease.tawk.help/article/http-api-request-configuration) or [Flow](https://apiease.tawk.help/article/flow-request-configuration) request.Once your configuration is complete, click the Save button at the top center of Shopify Admin. Your request is now ready to run based on the triggers you selected.

Source article: [How to Add Requests](https://apiease.tawk.help/article/how-to-add-requests).
