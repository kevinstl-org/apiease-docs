# Shopify access token

Follow these instructions to get your Shopify access token.

In your store admin console go to "Settings".Then go to "Apps and sales channels".Then click "Develop apps".Then click "Create an app".Give your app a name and click "Creat app".Configure the app with necessary permissions by clicking "Configure Admin API scopes" or "Configure Storefront API scopes" and choosing the permissions that you require. You must configure API scope in order to Install the app and get an access token.Click the "Install app" button at the top right.Click "Reveal token once". Copy and save your access token. You will only be able to see the access token one time so be sure to save it.

Source article: [Shopify access token](https://apiease.tawk.help/article/shopify-access-token).
